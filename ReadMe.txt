========================================================================
    WIN32 APPLICATION : 2GIStest
========================================================================
В каталоге Bin лежит текстовый файл и исполняемый файл приложения.

Без ошибок выполняются следующие наборы команд
-f Test.txt -m words -v "слово" - поиск последовательности символов "слово". Это необязательно может быть слово

-f Test.tхt -m checksum  - вычисление чексуммы

-h - мини хелп

Последовательность и набор команд в командной строке фиксированные. За этим следит конечный автомат CStateMashine.

Работой с файлами занимаеся класс CFileReader - возможно расширение на другие типы файлов кроме простых текстовых.

Выполненем команд занимается класс CDispatcher.

Работа классов CStateMashine и CDispatcher поностью  настраиваются.
